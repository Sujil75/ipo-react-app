# IPO Listing App – React

A responsive React application that displays upcoming and listed IPOs with a clean list-to-detail flow.

#### Features

- IPO List page with key highlights (dates, issue size, price band, lots)

- IPO Details page with structured IPO information and timeline

- Responsive design for web and mobile

- Data-driven UI using reusable list and detail schemas

- Consistent typography using Sora

#### Tech Stack

- React

- React Router

- CSS (responsive layouts)

- Static mock data (JSON)

#### Project Structure

- IPO List → High-level overview for quick scanning

- IPO Details → Deep dive into a single IPO using a shared id

- Logos and data are mapped via stable identifiers for clean routing

#### Design Philosophy

Built to mirror real-world fintech products: information-dense, predictable, and scalable.
Focus is on data modeling + UI clarity, not visual gimmicks.

### How to Run
```bash
npm install
npm start
```
